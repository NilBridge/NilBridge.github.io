原生命令

原生命令是NilBridge提供的一种用户与模块交互的方式

模块可以注册原生命令，从而使正则表达式可以调用

在群聊中发送/nbcmd xxx会返回执行结果，或者在控制台也可以直接执行

这里列举一些自带的命令

# stop

关闭NilBridge，一定要用这个关闭框架，否则数据不会被保存

# module

模块管理器，有如下子命令

# 加载一个模块

模块存放在modules文件夹中

module load <moduleName>

# 卸载一个模块

module unload <moduleName>

# 重载所有模块

module reload

# 列出所有已经加载的模块

module list

# qq

QQ管理器，有如下子命令

# 登录一个qq（不会被保存）

qq login <qq>

# 下线一个qq

qq logout <qq>

# 添加一个自动登录项（添加后不会自动登录，需要重启或者手动登录）

qq autologin add <qq>

# 移除一个自动登录项

qq autologin remove <qq>

# start_server

开启服务器

# 关闭一个服务器

start_server <服务器名称>

# stop_server

关闭服务器

# 开启一个服务器

stop_server <服务器名称>
