初始化模块

首先，在modules文件夹建立你以你模块名字为名字的文件夹

这里我用nb2-example为例子

#进入modules目录
cd modules
# 创建 nb2-example 文件夹、进入文件夹
md nb2-example &&　cd nb2-example
# 执行 npm init
npm init

然后npm就会问你一大堆东西，比如包名，作者，协议之类的

填好之后他会问你是否OK，输入yes就好

接着package.json就生成了

package.json作为模块的描述文件而存在，如果没有这个文件，模块管理器不会读取这个模块
